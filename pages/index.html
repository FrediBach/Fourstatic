Fourstatic helps you create static webpages that rule! It's a no-cms solution based on the Twig templating engine. We created something super simple and lightweight, that makes your work so much easier.

Some features:

  • Page themes with template inheritance and content macros
  • Smart data organization and structure with merge capability
  • Multiple data format support: JSON, YAML, XML, CSV and PHP
  • Support for external JSON API's
  • Dynamic relative urls for pages and media
  • Secure live development with merge functionality
  • Super easy thumbnails that are cached
  • Sass and Less support, shipped with mixin libraries
  • Markdown and CoffeeScript support
  • Fully cached for fast load times
  • Super easy one minute installation
  • Shipped with jKit macros and a nice boilerplate
  • Open source (MIT licensed)
  • And more ...

Objectives:

  • Avoid a steap learning curve
  • Work with the tools you know
  • Remove the dangers of live editing
  • Automate as much as possible
  • Support proven, modern, popular technology
  • Flexible and extendable

Requirements:

PHP 5
